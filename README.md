## Requirements

* node
* ruby / bundler

## Install

```sh
$ npm install; bower install; bundle install;
```

_Depending on your set up you may need to install `grunt-cli` & `bower` globally as well._

Install those via:

```sh
$ npm install -g grunt-cli bower
```

## Development

Want to run the project locally?

```sh
$ grunt serve
```
